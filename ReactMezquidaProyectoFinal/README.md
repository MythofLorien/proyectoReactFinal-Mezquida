Se usa la libreria npm de GH Pages. Pero para evitar problemas con el routeado de url las modificaciones que se realicen en el proyecto para ser reflejadas en el GH Page deberán ser las siguientes:<br>
1- npm run build<br>
2- npm run deploy<br>

- CartBuy
- CartItem
- CartItemContainer
- CartWidget
